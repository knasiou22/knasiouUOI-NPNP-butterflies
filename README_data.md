# Northern Pindos National Park, habitat & butterfly dataset


## Description of sampling and data 

In this study, we focused on mountainous communities of butterflies in Tymfi and Smolikas, in the Northern Pindos National Park. We selected 11 sampling sites on each mountain, in mountainous grasslands and forest clearings (1,250 – 1,940 m). We sampled butterflies in 22 transects (300 x 2.5 m: 3 visits) and recorded 10 habitat variables across three 25 m² plots per transect.
