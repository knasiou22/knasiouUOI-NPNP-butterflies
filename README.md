### Description
<!-- description: start -->
This study focuses on butterflies (Lepidoptera: Papilionoidea) and provides an integrated assessment of fine-scale environmental and seasonal filtering on multifaceted butterfly diversity, with the aim of addressing the following questions: (1) What is the value of the two mountains for butterfly conservation? (2) Are butterfly communities in forest clearings different from those in mountainous grasslands? (3) How is seasonality affecting butterfly communities and their diversity patterns? (4) Do taxonomic, functional and phylogenetic diversity spatially covary? (5) What are the effects of habitat variables on alpha diversity metrics and community composition? (6) How do habitat-trait interactions influence species abundances? 
<!-- description: end -->

### Details about the codes

Welcome to the repository for our study on the multifaceted mountainous butterfly diversity. This repository contains the R script used for the analyses. It is organized and documented to facilitate reproducibility and understanding of the analyses conducted in this study. Please follow the order of execution provided to replicate the results. The code proceeds step by step and includes all the necessary packages.
